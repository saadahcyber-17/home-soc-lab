# Home SOC Lab

## Overview

This project documents my personal Security Operations Center (SOC) lab built using Splunk Enterprise, Windows event logs, and custom Python-generated security logs.

The goal of this lab is to gain hands-on experience in:

- Security monitoring
- Log analysis
- Threat detection
- Alert investigation
- Detection engineering
- SOC workflows

- Authentication events
- Failed login attempts
- Brute force simulations
- User creation events
- Custom security log generation
