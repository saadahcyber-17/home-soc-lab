# Screenshots

This directory contains screenshots collected from the Home SOC Lab environment.

The screenshots provide visual evidence of lab setup, security monitoring activities, detections, dashboards, and incident investigations performed using Splunk.

## Folder Structure

### setup/

Screenshots related to environment setup and configuration.

Examples:

- Splunk installation
- Data ingestion
- Windows Event Log configuration
- Python log generator execution

### detections/

Screenshots of detection rules and alert results.

Examples:

- Failed Login Detection
- Brute Force Detection
- PowerShell Activity Detection

### dashboards/

Screenshots of Splunk dashboards and visualizations.

Examples:

- Authentication Dashboard
- Security Overview Dashboard
- Threat Monitoring Dashboard

### investigations/

Screenshots from security investigations and threat hunting activities.

Examples:

- Event analysis
- Search results
- Timeline reconstruction
- Incident investigations

## Purpose

The purpose of this directory is to document the practical work performed within the Home SOC Lab and demonstrate hands-on experience with:

- Security Monitoring
- Detection Engineering
- Incident Response
- Threat Hunting
- Splunk SIEM
